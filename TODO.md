
mapa com localização do entregador e envio da localização para o sistema. o sistema apos cada compra deve enviar uma notificação para o app dos entregadores que estiverem proximos.
ao aceitar a entrega o mapa devera automaticamente traçar uma rota visual entre o entregador e o local. para buscar a encomenda. quando o vendedor der o ok de despacho da encomenda, o cliente poderá ver a movimentação no mapa do entregador.
o stream da localização sera feito pelo rabbitmq em conjunto com o backend.
todos os mapas do front serão com openstreetmap/leaflet (e retire o nome do osm do canto inferior direito e tambem a bandeira da ucrania)
